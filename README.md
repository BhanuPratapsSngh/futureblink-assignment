# FutureBlink AI Flow App

## Tech Stack
- MongoDB, Express.js, Node.js, React, React Flow, OpenRouter API

## How to Run

### Backend
cd server
npm install
# Create .env file with:
# MONGO_URI=mongodb://localhost:27017/futureblink
# OPENROUTER_API_KEY=your_key_here
node index.js

### Frontend
cd client/Jobasses
npm install
npm run dev
